# 变更记录

## [v0.3.1] - 2026-06-26
### 修改
- 明确界面固定文案统一使用中文，不展示英文分区标题或英文说明。
- 明确组件、面板、日志区和结果区需要进一步紧凑化，减少不必要的大占位。

### 影响范围
- Spec / Design-Brief / UI / 前端测试

## [v0.3.0] - 2026-06-26
### 修改
- 全面重构桌面端界面方向：从单列卡片工具页升级为左侧导航、中间工作区、右侧状态面板的桌面应用 shell。
- 明确视觉参考 shadcn 的组件化、细边框、低圆角、浅色面板风格，以及 Mobbin / cobbin 式真实应用流和可扫读信息密度。
- 明确界面图标使用 SVG，禁止继续使用 emoji 作为按钮、导航或状态图标。
- 补充响应式要求：桌面三栏、平板两栏、手机单栏，关键点击目标不小于 44px。

### 影响范围
- Spec / Design-Brief / UI / 交互布局 / 可访问性 / 前端测试

## [v0.2.1] - 2026-06-26
### 修改
- 明确已配置下载保存目录时，下载操作直接执行，不再二次确认。
- 明确未配置下载保存目录时，下载前提示用户确认，并默认保存到用户桌面。
- 明确下载动作执行期间需要阻止重复触发，批量章节下载需要对重复章节 ID 去重。

### 影响范围
- Spec / UI / 桥接层 / 下载交互

## [v0.2.0] - 2026-06-25
### 新增
- 增加右上角设置入口中的下载保存目录选择需求。
- 增加本子封面和章节封面展示与预览需求。
- 增加从详情列表勾选多个章节并批量下载的需求。
- 增加手动输入多个章节 ID 后批量下载的需求。

### 修改
- 将保存目录配置从主工作台输入项调整为设置弹窗中的全局下载目录。
- 将下一步建议从 `/check` 调整为 `/ui`，因为本次变更影响页面结构和交互状态。

### 影响范围
- Spec / UI / 代码 / 桥接层 / Tauri 权限

## [v0.1.0] - 2026-06-25
### 新增
- 建立 JMComic Desktop 第一版产品需求基线。
- 明确 MVP 范围：查看详情、下载本子、下载章节、默认保存目录、日志与结果反馈、macOS 打包。
- 明确第一版不做登录、搜索、收藏夹、批量队列、内置阅读器和 AI 能力。
- 补齐页面清单、关键状态、用户流程、技术约束和验收标准。

### 影响范围
- Spec / UI / 代码检查 / 开发计划
